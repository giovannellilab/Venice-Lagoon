Data and code related to the Selci et al. manuscript on the microbial community structure of three different substrates from the Venice Lagoon. Sequences are available in the NCBI SRA archive with accession numbers PRJNA1119203 for the amplicon data.

The repository contains the following files:

- README.md (this readme file)
- Selci_et_al_Venice_Lagoon_16SrRNA_analysis.r (the R code to reproduce the figures)
- Venice_Lagoon_env_dataset.csv (the environmental dataset)
- a series of .csv and .rds files to replicate the figures and analyses reported in the paper
